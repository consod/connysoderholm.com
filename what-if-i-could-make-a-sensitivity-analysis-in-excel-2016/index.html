What if I could make a Sensitivity Analysis in Excel 2016?

Excel Conny Söderholm 06.06.2018
(Updated on 03.05.2023 )

Do you do Discounted Cash Flow (DCF) analyses, or calculate yearly fuel costs if both your consumption and cost vary? Calculating one or two changes is pretty easy, you change your variables and check your new results. But what do you do when you want to calculate a lot of these values at the same time? You use Excel Built-in What-if Analysis Data Table! Kudos to you if you spotted my pun in the headline by the way:) I will show you how to do a sensitivity analysis example based on fuel consumption and a yearly increase in fuel price. You can also find the completed Excel file here.

First, start by creating a table with some data and formulas as in the picture below. Set the Annual fuel cost formula in cell B4 to =B1*B2*(1+B3). Now we are ready to start the analysis.

The starting point for the sensitivity analysis.

Before you go to Data Table, select your table, from cell C6:K14. Click on the What-If analysis from the Data bar and then click on Data Table.

Select the Data Table from the What-If analysis.

Select the annual fuel consumption as the row input cell and the Annual fuel price increase as the Column input cell. Click Ok.

Selecting the Row and Column Cells.

If everything went Easy Street, you should have your table completed! For a sanity check, we can see that in cell F9 the result is $12.6M, as it should be with an increase of 5% in fuel price and a consumption of 20,000 tons. Now go and do some DCF's and nail that next project outlook!
