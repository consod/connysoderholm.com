Cruise Ship Data Visualization with Power BI

Creating excellent and informative charts is very challenging when you have large datasets. Excel crashes, and you get frustrated when you cannot get these final changes done. You can do the data visualization in Python with matplotlib or Plotly, but what if you want to get results quickly? The answer is to use Power BI, it is straightforward to use and creates a lot of helpful data visualizations out of the box. Read on to see an example of what you can do using Power BI with some cruise ship data.

What is Power BI then? According to Wikipedia: "Power BI is a business analytics service provided by Microsoft. It provides interactive visualizations with self-service business intelligence capabilities, where end users can create reports and dashboards by themselves, without having to depend on information technology staff or database administrators."

The Power BI statement is very accurate. I discovered that you could easily import various datasets and set different queries for your data. Ok, let's get started! The dataset I got covered cruise ship movement from 2003-2017. It also contained information about the ships such as length and Gross Tonnage. I downloaded the dataset from data.world which Brandon Telle had uploaded. The data was in csv file format. If you want to do some scraping yourself, Brandon has it on Github.

The data visualization

I started by analyzing the average ship length and how much Gross Tonnage the ships had. The average ship length for all vessels contained in the data set is 276m. The cruise line with the longest ships is Royal Caribbean. The total Gross Tonnage of the ships is 10576 thousand Gross Tonnes. Royal Caribbean leads the Gross Tonnage ratio also with 3M Gross Tonnage.

Funnel and Donut: Average ship length and Gross Tonnage visualization.

The next thing I looked at was the most popular ports (by how many ship visits it got) from 2003 to 2017. The most popular port was Fort Lauderdale with 15 600 visits.  The picture below is not showing all the ports. In Power BI, you can scroll down if you want to see more data.

Bar chart: The most visited port by all Cruise Ships 2003-2017.

With the data, it was easy to analyze the data with different parameters. The next thing I looked into was which Cruise Line had the most port visits. Carnival was first with almost 110 000 visits; Holland America Line came in second with 97 100 visits and Royal Caribbean third with 90 800 visits.

Multi-row card and Treemap: Cruise lines with the most port visits during 2003-2017.

The last thing I analyzed was which ship had the most port visits. Volendam won this contest.

Treemap: Ships with the most port visits.
