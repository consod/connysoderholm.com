Cell Data Validation in Excel

Sometimes you may want to limit your users' input options in a cell. You can write text in a nearby cell or input a comment, but from experience, users do what users do. You may also want to input some easily choosable alternatives for yourself, like an option to choose 1, 2 or 3 engines for example. Tag along to see how we can validate cell inputs easily in Excel.

Creating the spreadsheet

Let's open a new spreadsheet. Begin by writing "Number of Engines" in cell A1. Select cell B1 and write input the number 1. Write "SFOC" (Specific fuel consumption) in cell A2, "Power" and power in cell A3. Finally, input "Running time" in cell A4 and "Fuel consumed" in cell B5. Let's put an SFOC of 200 g/kWh in cell B2, an engine power of 4000 kW in B3 and a running time of 2000 hours in cell B4. Input the formula "=B1*B2*B3*B4/1E6" in cell B5. Do some formatting as you like in the cells. Your spreadsheet should now look similar as below. We now have formulas and input data to calculate total fuel consumption.

Inputs and formulas ready for data validation.

Validating the data

Select cell B1 and go to the data validation tab. Choose data validation → data validation from the data tools field as the picture below shows.

Data validation tool

Whole numbers and decimals

Click on settings and choose "Whole number",  "less than or equal to", maximum 6. Try to input a decimal number in cell B1. You will get a data validation error. Excel now only accepts whole numbers or integers in cell B1. Select cell B2 and go to data validation again. Select "Decimal" and values between 140 and 250. Write a letter in the cell. Excel does not accept the input, the same is true for cell A1.

Data validation error

List and custom validation

Go to cell E1 and input the number 1000. Write 2000 in cell E2, 3000 in cell E3 and so on until you reach 6000 in cell E6. Select cell B3 and select "List" from the Data Validation settings. Press the source button and select or write the range "=$E$1:$E$6". A drop-down menu has now appeared in cell B3 with the values 1000...6000 as options.

Data Validation with the List setting.

Finally, we will validate with a formula. Select cell F1 and write "Normal usage".  Select cell B4 and go to data validation and choose "Custom". Input the formula "=B1>0". Now go to cell B1 and input 0. Nothing happens. Go to cell B4 and press enter, now you get the error message. Be sure that the formula you write here evaluates to TRUE or FALSE.

Custom error messages

If you like, you can also tell the user what went wrong after input. Select the error alert and choose what to display. You can also choose to show a text box with information about what to input in the cell. There you go, I hope that you feel safer in how to inform your user of what to do in the spreadsheet the next time.
